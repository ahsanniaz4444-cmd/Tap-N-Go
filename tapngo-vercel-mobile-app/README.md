# Tap N Go Mobile POS App

A Vercel-ready full-stack Next.js mobile web app based on the Tap N Go proposal.

## Features
- Admin/Cashier login
- Mobile POS dashboard
- Product and inventory management
- QR/barcode SKU entry demo
- Sales billing and automatic stock update
- Reports and CSV export
- Vercel serverless API backend
