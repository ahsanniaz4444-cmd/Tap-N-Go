# Tap N Go - HTML CSS JavaScript Version

This folder contains a plain HTML/CSS/JavaScript conversion of the Tap N Go mobile POS app.

## Files
- `index.html` - main app file
- `style.css` - complete app styling
- `script.js` - login, dashboard, products, sales, reports, localStorage, CSV export, and charts

## How to run
1. Open the folder.
2. Double-click `index.html`.
3. Login using:
   - Email: `admin@tapngo.com`
   - Password: `admin123`

## Hosting on Vercel
1. Upload this folder to GitHub.
2. Import the repository in Vercel.
3. Framework preset: `Other` or `Static`.
4. No build command is required.
5. Output directory can be left blank/root.

## Notes
This is a front-end static version. Data is stored in browser `localStorage`, so it works without a backend server.
